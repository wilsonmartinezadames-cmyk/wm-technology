# 📚 Documentación API - WM TECHNOLOGY\n\n## Base URL\n```\nhttp://localhost:5000/api\n```\n\n## Authentication\nTodos los endpoints protegidos requieren:\n```\nAuthorization: Bearer {token}\n```\n\n---\n\n## 🔐 Autenticación (Auth)\n\n### POST /auth/register\nRegistrar nuevo usuario\n\n**Request:**\n```json\n{\n  \"name\": \"John Doe\",\n  \"email\": \"john@example.com\",\n  \"password\": \"password123\"\n}\n```\n\n**Response (201):**\n```json\n{\n  \"success\": true,\n  \"token\": \"eyJhbGc...\",\n  \"user\": {\n    \"_id\": \"123\",\n    \"name\": \"John Doe\",\n    \"email\": \"john@example.com\",\n    \"role\": \"user\",\n    \"createdAt\": \"2024-05-25T...\"\n  }\n}\n```\n\n### POST /auth/login\nIniciar sesión\n\n**Request:**\n```json\n{\n  \"email\": \"john@example.com\",\n  \"password\": \"password123\"\n}\n```\n\n**Response (200):**\n```json\n{\n  \"success\": true,\n  \"token\": \"eyJhbGc...\",\n  \"user\": { ... }\n}\n```\n\n### GET /auth/profile\nObtener perfil del usuario autenticado\n\n**Headers:**\n```\nAuthorization: Bearer {token}\n```\n\n**Response (200):**\n```json\n{\n  \"success\": true,\n  \"user\": { ... }\n}\n```\n\n### POST /auth/reset-password\nSolicitar reset de contraseña\n\n**Request:**\n```json\n{\n  \"email\": \"john@example.com\"\n}\n```\n\n**Response (200):**\n```json\n{\n  \"success\": true,\n  \"message\": \"Código de reset enviado a tu email\",\n  \"resetToken\": \"a1b2c3\" // Solo en desarrollo\n}\n```\n\n### POST /auth/verify-reset\nVerificar código y cambiar contraseña\n\n**Request:**\n```json\n{\n  \"email\": \"john@example.com\",\n  \"code\": \"a1b2c3\",\n  \"password\": \"newpassword123\"\n}\n```\n\n**Response (200):**\n```json\n{\n  \"success\": true,\n  \"message\": \"Contraseña actualizada correctamente\"\n}\n```\n\n---\n\n## 👤 Usuarios (Users)\n*Requiere autenticación*\n\n### GET /users/profile\nObtener perfil completo del usuario\n\n**Response (200):**\n```json\n{\n  \"success\": true,\n  \"user\": {\n    \"_id\": \"123\",\n    \"name\": \"John Doe\",\n    \"email\": \"john@example.com\",\n    \"phone\": \"+1234567890\",\n    \"company\": \"Tech Corp\",\n    \"avatar\": \"https://...\",\n    \"subscription\": { ... }\n  }\n}\n```\n\n### PUT /users/profile\nActualizar perfil del usuario\n\n**Request:**\n```json\n{\n  \"name\": \"John Updated\",\n  \"phone\": \"+1234567890\",\n  \"company\": \"New Company\"\n}\n```\n\n**Response (200):**\n```json\n{\n  \"success\": true,\n  \"message\": \"Perfil actualizado correctamente\",\n  \"user\": { ... }\n}\n```\n\n### POST /users/change-password\nCambiar contraseña\n\n**Request:**\n```json\n{\n  \"oldPassword\": \"oldpass123\",\n  \"newPassword\": \"newpass123\"\n}\n```\n\n**Response (200):**\n```json\n{\n  \"success\": true,\n  \"message\": \"Contraseña actualizada correctamente\"\n}\n```\n\n### GET /users/subscription\nObtener información de suscripción\n\n**Response (200):**\n```json\n{\n  \"success\": true,\n  \"subscription\": {\n    \"_id\": \"456\",\n    \"plan\": \"pro\",\n    \"status\": \"active\",\n    \"startDate\": \"2024-05-25T...\",\n    \"endDate\": \"2024-06-25T...\",\n    \"price\": 79,\n    \"features\": {\n      \"aiModels\": 10,\n      \"requestsPerMonth\": 10000,\n      \"storage\": 100\n    }\n  }\n}\n```\n\n---\n\n## 💬 Contacto (Contact)\n\n### POST /contact\nEnviar mensaje de contacto (Pública)\n\n**Request:**\n```json\n{\n  \"name\": \"Cliente\",\n  \"email\": \"cliente@example.com\",\n  \"subject\": \"Pregunta sobre planes\",\n  \"message\": \"Me gustaría saber más sobre el plan Enterprise\"\n}\n```\n\n**Response (201):**\n```json\n{\n  \"success\": true,\n  \"message\": \"Mensaje enviado correctamente. Nos contactaremos pronto.\",\n  \"contact\": {\n    \"_id\": \"789\",\n    \"name\": \"Cliente\",\n    \"email\": \"cliente@example.com\",\n    \"status\": \"new\",\n    \"createdAt\": \"2024-05-25T...\"\n  }\n}\n```\n\n### GET /contact\nObtener todos los mensajes de contacto (ADMIN)\n\n**Headers:**\n```\nAuthorization: Bearer {admin_token}\n```\n\n**Response (200):**\n```json\n{\n  \"success\": true,\n  \"count\": 15,\n  \"contacts\": [ ... ]\n}\n```\n\n---\n\n## 🔧 Health Check\n\n### GET /health\nVerificar estado del servidor\n\n**Response (200):**\n```json\n{\n  \"status\": \"Server is running\",\n  \"timestamp\": \"2024-05-25T13:58:00.000Z\",\n  \"environment\": \"development\",\n  \"apiVersion\": \"1.0.0\"\n}\n```\n\n---\n\n## ⚠️ Códigos de Error\n\n| Código | Descripción |\n|--------|-------------|\n| 200 | Éxito |\n| 201 | Creado |\n| 400 | Solicitud inválida |\n| 401 | No autenticado |\n| 403 | No autorizado |\n| 404 | No encontrado |\n| 500 | Error del servidor |\n\n---\n\n## 🧪 Testing con cURL\n\n### Registrar\n```bash\ncurl -X POST http://localhost:5000/api/auth/register \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"name\": \"John
