# 🚀 Guía de Instalación - WM TECHNOLOGY\n\n## Requisitos Previos\n\n- **Node.js** (v16 o superior) - [Descargar](https://nodejs.org/)\n- **npm** (incluido con Node.js)\n- **MongoDB Atlas** account - [Crear cuenta gratuita](https://www.mongodb.com/cloud/atlas)\n- **Git** - [Descargar](https://git-scm.com/)\n\n## Paso 1: Clonar el Repositorio\n\n```bash\ngit clone https://github.com/wilsonmartinezadames-cmyk/wm-technology.git\ncd wm-technology\n```\n\n## Paso 2: Configurar Frontend\n\n### 2.1 Instalar Dependencias\n\n```bash\ncd frontend\nnpm install\n```\n\n### 2.2 Crear archivo .env\n\nCopia el contenido de `.env.example` a un nuevo archivo `.env`:\n\n```bash\ncp .env.example .env\n```\n\nEdita `.env` con tus valores:\n\n```env\nVITE_API_URL=http://localhost:5000\nVITE_STRIPE_KEY=pk_test_xxxxx  # Tu clave pública de Stripe\nVITE_APP_NAME=WM TECHNOLOGY\n```\n\n### 2.3 Ejecutar Frontend\n\n```bash\nnpm run dev\n```\n\nEl frontend estará disponible en: **http://localhost:3000**\n\n## Paso 3: Configurar Backend\n\n### 3.1 Instalar Dependencias\n\nEn otra terminal:\n\n```bash\ncd backend\nnpm install\n```\n\n### 3.2 Configurar MongoDB Atlas\n\n1. Ve a [MongoDB Atlas](https://www.mongodb.com/cloud/atlas)\n2. Crea un nuevo cluster (gratis)\n3. Crea un usuario y contraseña\n4. Obtén la cadena de conexión (Connection String)\n5. Reemplaza `username`, `password` y `cluster` en la URL\n\n### 3.3 Crear archivo .env\n\nCopia el contenido de `.env.example` a un nuevo archivo `.env`:\n\n```bash\ncp .env.example .env\n```\n\nEdita `.env` con tus valores:\n\n```env\nPORT=5000\nMONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/wm-technology?retryWrites=true&w=majority\nJWT_SECRET=tu_secret_muy_seguro_aqui_123456\nJWT_EXPIRE=7d\nSTRIPE_SECRET_KEY=sk_test_xxxxx     # Tu secret key de Stripe\nSTRIPE_PUBLISH_KEY=pk_test_xxxxx    # Tu public key de Stripe\nNODE_ENV=development\nCLIENT_URL=http://localhost:3000\n```\n\n### 3.4 Ejecutar Backend\n\n```bash\nnpm run dev\n```\n\nEl backend estará disponible en: **http://localhost:5000**\n\nVerifica el health check: **http://localhost:5000/health**\n\n## Estructura del Proyecto\n\n```\nwm-technology/\n├── frontend/\n│   ├── public/                  # Archivos estáticos\n│   ├── src/\n│   │   ├── components/          # Componentes React\n│   │   │   ├── Header.jsx\n│   │   │   └── Footer.jsx\n│   │   ├── pages/               # Páginas\n│   │   │   └── Home.jsx\n│   │   ├── styles/              # CSS Global\n│   │   │   └── index.css\n│   │   ├── App.jsx              # Componente principal\n│   │   └── main.jsx             # Entry point\n│   ├── index.html\n│   ├── package.json\n│   ├── vite.config.js\n│   ├── tailwind.config.js\n│   └── postcss.config.js\n├── backend/\n│   ├── config/\n│   │   └── database.js          # Configuración MongoDB\n│   ├── routes/                  # Rutas API\n│   ├── controllers/             # Lógica de negocio\n│   ├── models/                  # Modelos MongoDB\n│   ├── middleware/              # Middlewares\n│   ├── server.js                # Servidor Express\n│   ├── package.json\n│   └── .env.example\n└── README.md\n```\n\n## Comandos Útiles\n\n### Frontend\n\n```bash\ncd frontend\nnpm run dev        # Ejecutar en desarrollo\nnpm run build      # Build para producción\nnpm run preview    # Previsualizar build\n```\n\n### Backend\n\n```bash\ncd backend\nnpm run dev        # Ejecutar con nodemon (recargar automático)\nnpm start          # Ejecutar servidor\n```\n\n## Pruebas Iniciales\n\n### Frontend\n\nAbre **http://localhost:3000** en tu navegador. Deberías ver:\n- ✅ Header con logo WM TECHNOLOGY\n- ✅ Hero Section con animaciones\n- ✅ Sección de Servicios\n- ✅ Footer\n\n### Backend\n\nHaz una petición GET a **http://localhost:5000/health**\n\n```bash\ncurl http://localhost:5000/health\n```\n\nDeberías obtener:\n\n```json\n{\n  \"status\": \"Server is running\",\n  \"timestamp\": \"2024-05-24T17:50:00.000Z\"\n}\n```\n\n## Solución de Problemas\n\n### Error: \"Cannot find module 'react'\"\n\n```bash\ncd frontend\nrm -rf node_modules package-lock.json\nnpm install\n```\n\n### Error: \"MongoDB connection failed\"\n\n- Verifica tu MONGODB_URI en `.env`\n- Asegúrate que el cluster esté activo en MongoDB Atlas\n- Comprueba que tu IP está en la whitelist\n\n### Puerto 3000 o 5000 en uso\n\n```bash\n# Cambiar puerto en frontend (vite.config.js)\nserver: {\n  port: 3001  // Cambiar a otro puerto\n}\n\n# Cambiar puerto en backend (.env)\nPORT=5001\n```\n\n## Próximos Pasos\n\n✅ Fase 1: Estructura base completada\n🔄 Fase 2: Implementar más páginas (Precios, Contacto, Auth)\n🔄 Fase 3: Crear APIs de autenticación\n🔄 Fase 4: Integrar Stripe y PayPal\n🔄 Fase 5: Dashboard administrativo\n\n## Recursos Útiles\n\n- [Documentación React](https://react.dev)\n- [Documentación Express](https://expressjs.com)\n- [Documentación MongoDB](https://docs.mongodb.com)\n- [Tailwind CSS](https://tailwindcss.com)\n- [Framer Motion](https://www.framer.com/motion)\n\n## Soporte\n\nSi tienes problemas, verifica:\n1.
